Once you have downloaded Python, a suitable IDE and Manim library you are ready to get started!
First, you need to import manim. Type from manim import * in your IDE to do this. Next, you will need to set up a Scene class. All your animations will each belong to one Scene class. Give your scene a descriptive name. for our first example, let's create a simple circle. So, we will type the following: class Draw_Circle(Scene): . Next, we will have to define a function called construct(self):. Every manim scene will follow this layout with minor changes depending on whether your scene will be 3D or 2D and so on.

Here comes the fun part. We can now create our circle and start animating! Lets call our circle, well, circle and so we type circle = Circle(). Manim has prebuilt classes of objects which we can use. Circles being one of them! Check out the documentation to see all the available shapes.

Lastly, we define the animation properties. First we tell the program to start the animation process. This is achieved by calling the self tag. We want the animation to first wait 1 sec at the start and then create the circle. So, we type self.wait() followed by self.play(Create(circle)). Let's see how the code looks in the IDE and what the video looks like.

To start the render process, type in the IDE terminal python scene.py Draw_Circle -pqh. Replace scene.py by the name of your python file. The last part -pqh means "play, quality high". This will output a 1080p video. Check out other render options in the documentation! 

                        
    from manim import *
    class Draw_Circle(Scene):
        def construct(self):
            circle = Circle()
            self.wait()
            self.play(Create(circle))
